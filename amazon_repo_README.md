## Overview

This repository is a **self-paced learning roadmap** for preparing for an Amazon Data Analyst interview. It covers:

✅ SQL fundamentals to advanced (CTEs, Window Functions, Subqueries)  
✅ Python data cleaning and analysis (Pandas, NumPy)  
✅ Tableau dashboard design and visualization  
✅ Real-world case studies and business problems  
✅ Interview tips and behavioral preparation  

**Target Audience:** Data analysts with basic SQL knowledge preparing for Amazon or similar FAANG interviews.

## Day-by-Day Breakdown

### SQL Module (Days 1-7)

**Day 1: Basic SELECT Queries**
- SELECT, FROM, WHERE clauses
- Column aliasing
- Sorting with ORDER BY
- Limiting results with LIMIT/TOP

**Day 2: Filtering & Aggregation**
- WHERE conditions (AND, OR, NOT)
- IN, BETWEEN, LIKE operators
- GROUP BY and HAVING
- Aggregate functions (COUNT, SUM, AVG, MIN, MAX)

**Day 3: Joins**
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- FULL OUTER JOIN
- CROSS JOIN
- Multi-table joins

**Day 4: Subqueries & CTEs**
- Scalar subqueries
- Correlated subqueries
- Subqueries in WHERE/FROM clauses
- Common Table Expressions (CTEs)
- Recursive CTEs

**Day 5: Window Functions**
- ROW_NUMBER(), RANK(), DENSE_RANK()
- LAG(), LEAD() for trend analysis
- SUM() OVER, AVG() OVER for running totals
- PARTITION BY for grouping
- NTILE() for bucketing

**Day 6: Advanced Queries**
- UNION and UNION ALL
- Set operations
- Complex problem solving
- Real-world scenarios

**Day 7: Performance Tuning**
- Query optimization strategies
- Index basics
- Execution plans
- Common pitfalls

## Setup Instructions

### Prerequisites
- SQL database (MySQL, PostgreSQL, or SQL Server)
- Python 3.8+
- Tableau Public (free) or Tableau Desktop
- Git

### Step 1: Clone Repository
```bash
git clone https://github.com/yourusername/Amazon-data-analyst-prep.git
cd Amazon-data-analyst-prep
```

### Step 2: SQL Setup
```bash
# Import sample data
mysql -u root -p < Data/sample_data.sql

# Or for PostgreSQL
psql -U postgres -d amazon_prep -f Data/sample_data.sql
```

### Step 3: Python Setup
```bash
# Create virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

### Step 4: Tableau Setup
- Download Tableau Public (free)
- Open `Tableau/Sample_Dashboard.twb`
- Connect to data files in `Tableau/Data_Files/`
