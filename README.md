> ## 📌 What's Inside
>
> This repo is a self-paced roadmap built to prepare for an Amazon Data Analyst role interview. It covers the exact skills tested in Amazon's technical rounds:
>
> - **SQL** — SELECT, filtering, aggregation, joins, subqueries, CTEs, window functions, performance tuning
> - - **Python** — Data cleaning, transformation, EDA, statistical analysis, A/B testing, cohort analysis, CLV
>   - - **Tableau** — Sales dashboards, customer analytics, operational KPIs
>     - - **Case Studies** — Sales analysis, customer retention, warehouse operations
>       - - **Behavioral** — Amazon Leadership Principles answered using the STAR method
>
